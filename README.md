# Fountain Blue Pools

Premium swimming pool construction, repairs, and renovations website for Fountain Blue Pools - Zimbabwe's trusted pool specialists.

## 🌊 Features

- **Modern React + Vite** - Fast, optimized build
- **Tailwind CSS** - Premium responsive design
- **WhatsApp Integration** - Direct customer contact
- **SEO Optimized** - Meta tags and semantic HTML
- **Mobile-First** - Fully responsive across all devices

## 🛠️ Services

- New Swimming Pool Construction
- Pool Repairs & Renovations
- Fibreglass Relining
- Marblelite Finishes
- Pump Repairs
- Maintenance Plans

## 🚀 Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## 📦 Tech Stack

- React 18
- Vite 5
- Tailwind CSS 3
- React Router
- Lucide Icons

## 🌐 Deployment

Ready for deployment on Vercel, Netlify, or any static hosting.
